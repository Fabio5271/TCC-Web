Sistema de Análise de Ocorrência de Doenças Respiratórias no Distrito Federal Baseado em Dados Governamentais

Introdução

O sistema realiza a análise de ocorrência de doenças respiratórias no Distrito Federal, utilizando dados governamentais abertos.

A pesquisa busca compreender como fatores climáticos e ambientais influenciam a incidência dessas doenças, sobretudo durante o período seco.

A Análise de Dados

O sistema integra dados de fontes oficiais, como Fiocruz, DATASUS, CEMADEN e INMET, aplicando técnicas de Ciência de Dados para identificar padrões e correlações. Fundamentado em estudos que relacionam clima, poluição e saúde pública, o projeto adota métodos de análise estatística e aprendizado de máquina para gerar modelos preditivos e visualizações interativas.

Objetivos

Espera-se que o sistema contribua para a vigilância epidemiológica e o planejamento estratégico em saúde pública, fortalecendo a integração entre tecnologia e políticas preventivas.
